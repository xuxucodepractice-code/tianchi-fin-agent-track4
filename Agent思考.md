# 选项 D 思考

## 选项

D. 平安富鸿金生无论何种投保方式，均不允许保单贷款

## 要验证的判断点

- 主体是否对应：平安富鸿金生。
- 原文是否支持“无论何种投保方式”这个绝对范围。
- 原文是否只是限制“按个人养老金制度投保”的情况。
- 如果原文只限制特定投保方式，D 是否属于过度概括。

## 使用的文件

```text
public_dataset_upload/raw/insurance/16.pdf
```

## 搜索关键词

- 保单贷款
- 个人养老金制度
- 投保方式
- 不接受保单贷款申请
- 现金价值权益

## 搜索过程

1. 首要搜索关键词是“保单贷款”。
2. 根据目录和关键词搜索，定位到第六章“现金价值权益”中的 6.2“保单贷款”。
3. 因为 D 的说法是“无论何种投保方式，均不允许保单贷款”，所以重点不是只找“不接受保单贷款申请”，还要看原文是否存在一般情况下可以申请保单贷款的表述。
4. 在 6.2 中同时看到一般性允许保单贷款的表述，以及针对个人养老金制度投保的限制表述，因此需要区分“通常情况”和“特定条件”。

## 找到的原文证据

- 依据一：“经被保险人书面同意，您可申请保单贷款功能。”
- 依据二：“在本合同有效期内，经我们审核同意后您可办理保单贷款。”
- 对照依据：“若您按照个人养老金制度(10.14)投保并订立本合同的，我们不接受保单贷款申请。”

## 证据位置

```text
PDF-16
第六章 现金价值权益
6.2 保单贷款
```

## 我的判断

判断 D 错误。

理由是：

```text
D 的说法是“无论何种投保方式，均不允许保单贷款”，这是一个绝对化表述。

但原文 6.2 中存在一般性允许保单贷款的表述：
“您可申请保单贷款功能”
“您可办理保单贷款”

同时，原文只对“按照个人养老金制度投保并订立本合同”的情况作出限制：
“我们不接受保单贷款申请”。

因此，原文更像是：
一般情况下可以申请保单贷款；
但如果按个人养老金制度投保，则不接受保单贷款申请。

所以 D 把特定条件下的不允许，扩大成“无论何种投保方式均不允许”，属于过度概括。
```

## 仍不确定的地方

需要在最终证据摘录时确认三句话是否都来自同一个 6.2 条款，并且上下文没有其他相反限制。

但就当前找到的证据而言，D 的“无论何种投保方式”不被原文支持。
