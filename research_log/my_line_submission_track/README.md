## 合规红线（贯穿所有版本）

- 正式答题只用 **Qwen 系列**（阿里云百炼 OpenAI 兼容接口），不得用其他模型替代。
- **无 embedding、无小模型、不联网**。检索/补查纯 BM25。
- 所有模型调用的 token 都计入 TokenScore；API 缓存省钱但不减 TokenScore → 必须真正少喂。
- deepseek 网关(deepseek-v4-pro) 仅用于**开发期验证方案方向**，非正式提交模型。

## 参考文献血统（详见 改进策略.md 第十六章）

- 检索地基：**BM25**(Okapi)
- 压缩层：**RECOMP**(arXiv:2310.04408) + **LLMLingua/LongLLMLingua**(微软)
- agent循环：**Self-RAG**(arXiv:2310.11511) + **CRAG**(arXiv:2401.15884)
- 按需/增量：**Adaptive-RAG**(arXiv:2403.14403) + **FLARE**(arXiv:2305.06983) + **DRAGIN**(arXiv:2403.10081)
- 保守动作：自研（把"病=多选漏选"编进 agent 动作空间）
