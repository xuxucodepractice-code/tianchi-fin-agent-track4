## What This Step Teaches

这一步完成了从工具辅助到模型判断的小闭环：

```text
工具负责找证据。
人负责整理 evidence pack。
prompt 负责约束模型只能基于证据判断。
模型负责逐项判断 A/B/C/D。
人负责复核模型输出是否可靠。
```

关键收获：

```text
1. 模型不能直接裸答，必须被证据约束。
2. Evidence pack 的质量会直接影响模型判断质量。
3. Prompt 需要明确区分“支持”“反驳”“证据不足”。
4. 多选题适合先逐项判断，再汇总最终答案。
5. 条件词和范围词是推理判断题的重要风险点。
```
