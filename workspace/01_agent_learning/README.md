# 01 Agent Learning

目标：沿着原 AI Agent 学习计划循序渐进，把比赛任务拆成可以逐步掌握的小台阶。

## 应放内容

- 人工解题记录
- Agent 裸循环实验
- 工具调用实验
- 课程和论文笔记
- 状态图、trace、调试记录
- 对 Agent、workflow、tool use、memory、guardrails 的理解笔记

## 推荐子目录

```text
manual_solving/     # 人肉解题记录
minimal_loop/       # 零框架 Think-Act-Observe 小实验
tool_calling/       # 文档读取、检索、计算、Token 统计等工具实验
state_graph/        # LangGraph 或显式状态机练习
notes/              # 课程、论文、博客笔记
```

## 通关目标

能解释并实现：

```text
题目 -> 找材料 -> 找证据 -> 判断选项 -> 输出答案
```

并理解这些动作如何逐步变成 Agent 系统。
