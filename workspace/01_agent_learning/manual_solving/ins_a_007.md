## Option Decomposition

### A. 平安智盈金生允许保单贷款，最高为现金价值的80%

这个选项至少包含几个判断点：

- 主体是否对应：平安智盈金生。
- 是否允许保单贷款。
- 如果允许，贷款上限是否与“现金价值”相关。
- 贷款上限是否为 80%。

判断逻辑：

```text
先确认平安智盈金生是否允许保单贷款。
如果不允许，后面的“最高为现金价值的80%”就没有意义。
如果允许，再确认上限基准是不是现金价值，以及比例是不是80%。
```

### B. 国寿增益宝允许保单贷款，最高为现金价值扣除欠款后的80%

这个选项至少包含几个判断点：

- 主体是否对应：国寿增益宝。
- 是否允许保单贷款。
- 贷款上限是否基于现金价值。
- 是否需要先扣除欠款。
- 贷款上限比例是否为 80%。

判断逻辑：

```text
先确认国寿增益宝是否允许保单贷款。
如果允许，再确认原文表述是不是“现金价值扣除欠款后的80%”。
重点关注“现金价值”“扣除欠款”“最高”“80%”这些限定词。
```

### C. 平安富鸿金生若按个人养老金制度投保，则不允许保单贷款

这个选项至少包含几个判断点：

- 主体是否对应：平安富鸿金生。
- 条件是否对应：按个人养老金制度投保。
- 在这个条件下是否不允许保单贷款。
- 原文说的对象是否确实是“保单贷款”。

判断逻辑：

```text
这个选项是有条件判断。
它不是说平安富鸿金生永远不能保单贷款，而是说“若按个人养老金制度投保”时不允许。
所以要重点查个人养老金制度投保方式和保单贷款限制之间的关系。
```

### D. 平安富鸿金生无论何种投保方式，均不允许保单贷款

这个选项至少包含几个判断点：

- 主体是否对应：平安富鸿金生。
- 条件范围是否是“无论何种投保方式”。
- 是否完全不允许保单贷款。
- 原文说的对象是否确实是“保单贷款”。

判断逻辑：

```text
这个选项比 C 更绝对。
C 是有条件的不允许，D 是无条件、所有投保方式都不允许。
需要判断原文是不是支持这么绝对的说法。
```
