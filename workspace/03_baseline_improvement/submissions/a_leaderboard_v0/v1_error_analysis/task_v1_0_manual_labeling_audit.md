## 2. 总体发现

- **5 道 tf 的初步判断全部指向 A（正确）**——v0 全 A 在这 5 题上可能碰巧全对。这个样本有选择偏差（全部取自 low_confidence 清单），但它修正了审查报告的预期：tf 全 A 的损失可能没有"8-10 题"那么大，**需要人工再核 lc 之外的部分 tf 题**（如 fc_a_003/006/010 这类"正常判定"题）才能下结论。
- 但 tf 链路确实是坏的，只是坏法不同：5 题里 2 题（res_a_003、fin_a_013）出现 **judgment 标签与 rationale 内容自相矛盾**（rationale 说"原命题为真"，标签却打了 refute/support 的错位组合），答案对是 fallback 撞对的，不是链路对。
- 发现一个**新的 Task 3 解析缺陷**：`csrc_0009_att1`（上市公司信息披露管理办法）的 PDF 抽取结果是**逐字换行**（"定\n期\n报\n告"），21/21 个 chunk 全部如此，任何多字词都无法被 BM25 命中。该文档被 reg_a_006、reg_a_015、reg_a_018 三道题引用——这三题的检索对该文档全盲。全库扫描确认只有这一个文档受影响。
