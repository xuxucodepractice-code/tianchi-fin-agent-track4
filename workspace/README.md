# Workspace

这个目录承载比赛过程中的学习、baseline 理解和优化实验。

根目录职责划分：

```text
docs/                  # 规则、赛制、路线图和事实沉淀
public_dataset_upload/ # 官方公开数据和训练材料
workspace/             # 我们自己的学习过程、baseline 拆解和实验记录
```

三阶段工作区：

- [01_agent_learning](01_agent_learning/README.md)：循序渐进学习 Agent，从人肉解题到最小循环、工具调用、状态图和评估。
- [02_baseline_understanding](02_baseline_understanding/README.md)：理解 baseline，拆解数据流、模块职责、单题复现路径和失败模式。
- [03_baseline_improvement](03_baseline_improvement/README.md)：改进、强化或重构 baseline，记录实验、错题、prompt、检索策略和 Token 优化。
